# アーキテクチャ設計書 (AI Chord Tracker)

## 概要
本システムは、音声ファイルからオーディオストリームを読み込み、機械学習モデルを用いて和音（コード）進行を予測、ならびにMIDIへのエクスポートを行うデスクトップアプリケーションである。

## モジュール構成
1. **ユーザーインターフェース (UI)**
   - `main_window.py`: メインウィンドウ管理、再生コントロール
   - `timeline_widget.py`: コード結果の可視化とインタラクション
2. **音声処理 (Audio Processing)**
   - `audio_processor.py`: `librosa`を用いた音声読み込み、CQT（定Q変換）によるスペクトログラム作成、拍子（テンポ）推定、キーの判定やモデルの出力からタイムラインへの変換。
3. **推論モデル (ML Model)**
   - `chord_model.py`: TensorFlow/Kerasを用いた和音予測。HuggingFaceから自動ダウンロード。（ローカル実行）
4. **エクスポート機能 (Export)**
   - `midi_export.py`: 予測された和音タイムライン（テキスト）を解析し、`mido`ライブラリを用いてMIDIイベントに変換してファイルへ書き出し。

## テストアーキテクチャ
- **フレームワーク**: `pytest`
- **対象**: 音声処理ロジック・推論データハンドリング・MIDIエクスポートのロジックを対象とする単体テスト（Unit Test）。
- **指針**: 音声処理ロジックの正確性を担保するため、数学的変換（CQTのパディングなど）、和音名からMIDIノートへの変換などのエッジケースを検証する。重いモデルのダウンロードや実行、および外部ファイル依存箇所はモック（Mock）化し、ローカル開発環境で高速に実行できるようにする。
